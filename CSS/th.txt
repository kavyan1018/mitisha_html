types of css : 
    1. Inline CSS :- 
        - Inline CSS is used to apply a unique style to a single HTML element.
        - To use inline CSS, the style attribute is added to the relevant tag.
        - The style attribute can contain any CSS property.
        - Inline CSS is not recommended for large projects as it can make the code difficult to read and maintain.

    2. Internal CSS
        - Internal CSS is used to define styles for a single HTML document.
        - The internal CSS is defined within the <style> tag in the <head> section of the HTML document.
        - Internal CSS is useful for styling a single page without affecting other pages.
        
    3. External CSS
        - External CSS is used to define styles for multiple HTML documents.
        - The external CSS is defined in a separate .css file and linked to the HTML document using the <link> tag.
        - External CSS is the most efficient way to style a website as it allows for easy maintenance and updates.
